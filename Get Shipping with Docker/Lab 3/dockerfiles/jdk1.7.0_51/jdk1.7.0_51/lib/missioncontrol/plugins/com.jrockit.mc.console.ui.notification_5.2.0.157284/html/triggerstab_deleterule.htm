Deleting a Trigger Rule

Deleting a trigger rule removes it completely from the Trigger Rules list; it cannot be used for the running application without being recreated, as described in Adding a New Trigger Rule Using the Add Wizard.

To delete a trigger rule:

  • Select the rule on the Trigger Rules panel and click Delete.

You must sure that you want to delete the rule, because you do not get any confirmation before the rule is deleted.
