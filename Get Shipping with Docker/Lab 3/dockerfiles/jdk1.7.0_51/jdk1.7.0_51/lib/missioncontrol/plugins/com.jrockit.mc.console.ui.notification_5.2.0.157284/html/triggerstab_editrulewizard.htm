Editing a Trigger Rule Using the Edit Wizard

Editing a trigger rule with the Edit wizard allows you to change the properties of a rule without having to create a new one. The process of editing a rule by using the Edit wizard follows a similar sequence as the process described in "Adding a New Trigger Rule Using the Add Wizard".

To edit a rule by using the Edit wizard:

  1. Select the rule you want to edit.

  2. Click Edit.

    The Edit <rule name> wizard appears (where <rule name> is the name of the rule as it appears on Trigger Rules list.

  3. Click Next until you arrive at the dialog box that contains the property you want to edit.

  4. After you have edited the necessary properties, click Finish.

    All edits will appear on the affected property panels on the Triggers tab.
