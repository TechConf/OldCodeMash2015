Getting Familiar with the Triggers Tab

When you open the Triggers tab, you can see some default trigger rules in the Trigger Rules panel. In addition to the rules, you can see buttons such as Add, Edit, Import, Export, and Rename to manage the rules.

The Triggers tab is divided into the following sections:

  • Trigger Rules: Lists all the rules that you have added and lets you add, delete and edit rules.

  • Trigger Condition: Lets you browse for the attribute on which the rule should trigger.

  • Trigger Action: Selects the action for the Java Mission Control Client to perform when the rule triggers an event.

  • Trigger Constraints: Sets the constraints of the rule, for example, it limits the action to specific days of the week or specific times of day.
