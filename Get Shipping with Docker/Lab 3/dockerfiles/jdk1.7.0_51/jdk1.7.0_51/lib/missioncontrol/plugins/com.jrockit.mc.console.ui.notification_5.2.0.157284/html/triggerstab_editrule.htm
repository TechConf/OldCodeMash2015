Editing a Trigger Rule on the Triggers Tab

When a rule appears for the first time, three new panels open on the Triggers tab. These panels contain the same information you entered for the rule when following the steps in "Adding a New Trigger Rule Using the Add Wizard". You can use these panels to edit an existing rule directly on the Trigger tab. These panel appear as soon as you select a trigger rule under the Trigger Rules panel.

To edit a trigger rule directly on the Triggers tab:

  1. Click on the trigger rule you want to edit.

  2. In Trigger Condition (Figure: Trigger Condition Panel), set the properties that define when the rule will be triggered; for example, the threshold at which the rule is invoked and for how long the triggering condition should be sustained before invoking the rule. Table: Set Condition for Triggering Field Descriptions describes the fields on this panel.

    Description of "Trigger Condition Panel"

  3. In the Trigger Action panel (Figure: Trigger Action Panel), set the type of action you want your rule to perform when your rule is triggered. You have a choice of these options:

    Trigger Action Panel


    Description of "Trigger Action Panel"

    Depending on the action you select, you might have to enter additional data.

  4. In the Trigger Constraints panel (Figure: Trigger Constraints Panel), set any constraints you want on the rule. This is an optional step that allows you to limit when a trigger action is activated; for example, you might want your trigger action to be activated only every other day of the week or during specific hours of the day.

    Trigger Constraints Panel


    Description of "Trigger Constraints Panel"

    You can select from these options:

    • Date span sets a range of dates.

    • Day of week sets which days of the week you want the trigger action to be activated. If you select this option, you must specify which days you want it to be activated.

    • Time span sets the time period.
