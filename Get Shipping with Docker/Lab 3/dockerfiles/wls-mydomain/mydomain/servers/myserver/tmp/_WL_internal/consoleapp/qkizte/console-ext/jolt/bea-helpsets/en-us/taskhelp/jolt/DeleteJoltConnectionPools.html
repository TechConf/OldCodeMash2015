Delete Jolt connection pools


Note: Deleting a connection pool permanently removes it from the domain's configuration. If you only want to temporarily stop applications from accessing this connection pool, un-target it from all servers. See: Target Jolt connection pools

To delete Jolt connection pools:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select Jolt Connection Pools.
  3. On the Jolt Connection Pools page, click the name of a connection pool.
  4. Click Delete and confirm the deletion.
  5. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
