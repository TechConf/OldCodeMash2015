Use this page to monitor Coherence clusters associated with this application.

This table shows run-time information for a Coherence cluster associated with a particular application.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.

Name Description
Cluster Name

Name of this Coherence cluster.
Server

The name of the WebLogic Server instance hosting this Coherence cluster node.
Cluster Size

Coherence clusters can consist of nodes formed by applications or modules as well as stand-alone Coherence cache servers. The cluster size shows the number of such nodes currently running in this Coherence cluster.
Members

Identifiers for the available Coherence cluster members.
License Mode

The license mode for this Coherence cluster. Possible values are Evaluation, Development, or Production.
Version

The Coherence cluster version.
