Configure address providers


An address provider specifies the TCP listener address (IP, or DNS name, and port) for a proxy service.

To configure address providers:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane of the Console, select Environment > Coherence Clusters.
  3. In the Coherence Clusters table, select the Coherence cluster for which you want a configure an address provider.
  4. Select the Configuration: Address Providers page.
  5. To create a new address provider, click New, or select the name of an existing address provider.
  6. To create a new socket address for your Coherence address provider, click New or select the name of an existing socket address. To configure a new socket address:
    1. In Name, specify the name for your new Coherence cluster socket address.
    2. In Listen Address, specify the listen address for your socket address.
    3. In Listen Port, specify the listen port for your socket address.
    4. Click OK.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
