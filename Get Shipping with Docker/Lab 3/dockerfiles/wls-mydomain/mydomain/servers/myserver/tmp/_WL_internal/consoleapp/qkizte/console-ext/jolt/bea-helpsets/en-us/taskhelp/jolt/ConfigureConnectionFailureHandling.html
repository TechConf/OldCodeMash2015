Configure Jolt addresses


To specify the Jolt servers that the connection pool connects to:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select Jolt Connection Pools.
  3. On the Jolt Connection Pools page, click the name of a connection pool.
  4. Click the Configuration: Addresses tab.
  5. On the Addresses page, in Primary Addresses, enter one or more addresses of the preferred Jolt Service Listeners (JSLs).

    For information about the format of addresses, see Configuration Options.

  6. (Optional) If you want the connection pool to use some other set of JSLs if connections to the peferred JSLs fail, enter one or more addresses in Failover Addresses.
  7. Click Save.
  8. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
