WTC Server: Configuration: Redirections

Column Display     Related Tasks     Related Topics

Redirections are used to configure a one-to-one connection between the JMS interface and Tuxedo application environment.

This page summarizes the redirections that have been configured for the current WTC server.

Note:
You must create a queuing bridge configuration before you can create a redirections configuration.
