Queuing Bridge: Priority Mapping

Configuration Options     Related Tasks     Related Topics

Use this page to define the priority mapping for this Tuxedo queuing bridge.

Use priority mapping to map priorities between the JMS and Tuxedo. JMS has ten priorities (0 - 9); Tuxedo/Q has 100 priorities (1 - 100). There are two mapping directions: JMS to Tux and Tux to JMS.

Configuration Options

Name Description
JMS To Tux Priority Map

The mapping of priorities when mapping from JMS to Tuxedo.
Tux To JMS Priority Map

The mapping of priorities when mapping from Tuxedo to JMS.
