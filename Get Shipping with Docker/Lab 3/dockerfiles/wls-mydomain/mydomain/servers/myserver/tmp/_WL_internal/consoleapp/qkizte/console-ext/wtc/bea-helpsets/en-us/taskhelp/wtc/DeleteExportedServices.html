Delete exported services


Exported services provide information about how to provide Java application services to remote Tuxedo application environments. To delete an exported service:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to delete exported services.
  4. Select Configuration > Exported.
  5. On the WTC Exported Services page, select the checkbox next to the exported service you want to delete.
  6. Click Delete.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
