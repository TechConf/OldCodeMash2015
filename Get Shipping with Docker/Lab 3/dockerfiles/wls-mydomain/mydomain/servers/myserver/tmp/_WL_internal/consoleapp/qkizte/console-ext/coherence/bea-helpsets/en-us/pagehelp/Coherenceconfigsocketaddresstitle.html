Coherence Cluster: Configuration: Socket Address

Configuration Options     Related Tasks     Related Topics

Use this page to configure a Socket Address for this Coherence Address Provider.

Address providers allow Proxy service, remote cache, and remote invocation services to reference configured TCP socket address combinations by name instead of explicitly embedding an addresses in a Cache Configuration file.

Configuration Options

Name Description
Name

A unique name for this Coherence cluster address.
Listen Address

The IP address.

MBean Attribute:
CoherenceSocketAddressBean.Address

Changes take effect after you redeploy the module or restart the server.

Listen Port

The listen port.

MBean Attribute:
CoherenceSocketAddressBean.Port

Minimum value: 1

Maximum value: 65535

Changes take effect after you redeploy the module or restart the server.
