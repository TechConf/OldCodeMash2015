Use this page to configure properties for this Cache Configuration.

Configuration Options

Name Description
Name

Name of this Cache Configuration.

JNDI Name

The JNDI name this resource will be bound to.

MBean Attribute:
CoherenceCacheConfigMBean.JNDIName

Changes take effect after you redeploy the module or restart the server.

Cache Configuration File

Return the cache configuration file being used by this CoherenceCacheConfigMBean.

MBean Attribute:
CoherenceCacheConfigMBean.CacheConfigurationFile

Changes take effect after you redeploy the module or restart the server.

Import Configuration File

Check this field if you want to import the cache configuration file.
