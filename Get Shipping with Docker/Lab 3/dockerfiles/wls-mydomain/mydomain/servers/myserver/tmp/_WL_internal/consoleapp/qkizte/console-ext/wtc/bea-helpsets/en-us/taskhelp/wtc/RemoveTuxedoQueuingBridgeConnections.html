Delete queuing bridge connections


A Tuxedo queuing bridge provides a bidirectional JMS interface that allows WTC applications to communicate with Tuxedo application environments. To remove a Tuxedo queuing bridge:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to delete queuing bridge connections.
  4. Select Configuration > Queuing Bridge.
  5. On the WTC Tuxedo Queuing Bridges page, select the checkbox next to the queuing bridge you want to delete.
  6. Click Delete.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
