Coherence Cluster: Configuration: Cache Configurations

Column Display     Related Tasks     Related Topics

This page displays all the cache configurations for a Coherence cluster.

Cache configurations provide the ability to override the cache configuration file within a Grid Archive (GAR) with a different file. A cache configuration can be targeted at a Managed Server or a cluster. This allows an application to use of all the Coherence artifacts within a GAR, yet utilize a separate cache configuration file. An example of this would be to enable Proxy Service for Coherence*Extend connections to a cluster for a specific Managed Server or cluster.
