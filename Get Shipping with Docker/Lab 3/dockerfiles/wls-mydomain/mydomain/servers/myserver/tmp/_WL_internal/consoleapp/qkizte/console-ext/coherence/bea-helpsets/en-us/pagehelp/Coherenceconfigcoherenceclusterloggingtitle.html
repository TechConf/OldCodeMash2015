Coherence Cluster: Configuration: Logging

Configuration Options     Related Tasks     Related Topics

Use this page to configure logging properties for this Coherence cluster.

Configuration Options

Name Description
Enabled

Specifies whether custom Coherence logging is enabled.

MBean Attribute:
CoherenceLoggingParamsBean.Enabled

Changes take effect after you redeploy the module or restart the server.

Logger Name

The logger name for Coherence logs.

MBean Attribute:
CoherenceLoggingParamsBean.LoggerName

Changes take effect after you redeploy the module or restart the server.

Message Format

The Coherence logs message format.

MBean Attribute:
CoherenceLoggingParamsBean.MessageFormat

Changes take effect after you redeploy the module or restart the server.
