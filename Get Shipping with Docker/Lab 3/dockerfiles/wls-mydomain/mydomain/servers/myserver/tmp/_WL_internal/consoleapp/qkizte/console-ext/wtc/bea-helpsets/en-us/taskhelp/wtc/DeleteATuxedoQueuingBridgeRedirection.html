Delete Tuxedo queuing bridge redirections


Redirections are used to configure a one-to-one connection between the JMS interface and Tuxedo application environment. To delete a redirection:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to delete Tuxedo queuing bridge redirections.
  4. Select Configuration > Redirections.
  5. On the WTC Redirections page, select the checkbox next to the redirection you want to delete.
  6. Click Delete.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
