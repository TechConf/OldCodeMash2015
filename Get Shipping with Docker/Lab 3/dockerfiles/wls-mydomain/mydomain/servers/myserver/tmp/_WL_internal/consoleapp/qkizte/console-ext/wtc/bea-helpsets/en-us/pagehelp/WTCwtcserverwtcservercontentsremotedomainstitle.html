WTC Server: Configuration: Remote APs

Column Display     Related Tasks     Related Topics

Remote Tuxedo access points provide configuration information to connect a WTC server to available remote Tuxedo domains.

This page summarizes the remote Tuxedo access points that have been configured for the current WTC server.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.

Name Description
Name

The unique name used to identify this remote Tuxedo access point. This name should be unique for all local and remote Tuxedo access points defined within a WTC Service. This allows you to define unique configurations having the same Access Point ID.

MBean Attribute:
WTCRemoteTuxDomMBean.AccessPoint

Access Point ID

The connection principal name used to identify this remote Tuxedo access point when attempting to establish a session connection to local Tuxedo access points.

MBean Attribute:
WTCRemoteTuxDomMBean.AccessPointId

Local Access Point

The local domain name from which this remote Tuxedo domain is reached.

MBean Attribute:
WTCRemoteTuxDomMBean.LocalAccessPoint
