Cluster: Configuration: Coherence

Configuration Options     Related Tasks     

Use this page to associate this WebLogic Server cluster with a Coherence cluster. After this WebLogic Server cluster is associated with a Coherence cluster, configure Coherence cluster member properties as required. All server instances that are managed together as part of this WebLogic Server cluster share the same Coherence cluster configuration

Configuration Options

Name Description
Local Storage Enabled

Specifies whether Local Storage is enabled

MBean Attribute:
CoherenceTierMBean.LocalStorageEnabled

Changes take effect after you redeploy the module or restart the server.

Coherence Web Local Storage Enabled

Specifies whether Local Storage is enabled for the Coherence*Web cluster tier

MBean Attribute:
CoherenceTierMBean.CoherenceWebLocalStorageEnabled

Changes take effect after you redeploy the module or restart the server.

Coherence Cluster

Coherence Cluster associated with this cluster.

MBean Attribute:
ClusterMBean.CoherenceClusterSystemResource

Changes take effect after you redeploy the module or restart the server.
