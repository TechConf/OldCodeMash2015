Coherence Cluster: Targets

Configuration Options     Related Tasks     Related Topics

Use this page to select the servers or clusters on which you want to deploy this Coherence cluster configuration.

In addition to these explicitly targeted servers and clusters, this Coherence Cluster may be associated with one or more Server Templates. Servers and clusters which subsequently use those Server Templates will also have Coherence Cluster features available.

Configuration Options

Name Description
Independent Servers

Lists the servers in your configuration on which you can deploy the Coherence cluster. Select servers and click Save.
Clusters

Lists clusters and comprising servers on which you can deploy the Coherence cluster. Select the cluster on which you want to deploy the Coherence cluster and click Save. In almost all cases, you should deploy the Coherence cluster to an entire cluster, not individual servers in a cluster.
