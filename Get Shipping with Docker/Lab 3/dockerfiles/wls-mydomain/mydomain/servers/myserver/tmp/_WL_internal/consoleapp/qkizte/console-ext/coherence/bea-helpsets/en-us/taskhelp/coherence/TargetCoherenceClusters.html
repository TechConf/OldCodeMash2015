Target Coherence clusters


Target Coherence clusters to WebLogic Server instances or clusters that host applications that need access to Coherence data caches and services.

To target Coherence clusters:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane of the Console, expand Environment and select Coherence Clusters.

    The Summary of Coherence Clusters page displays the Coherence cluster configurations that have been created in this domain.
  3. Click the name of the cluster for which you want to configure targets.
  4. Select Members.
  5. Select the servers or clusters on which you want to deploy this Coherence cluster configuration.
  6. Click Save.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
