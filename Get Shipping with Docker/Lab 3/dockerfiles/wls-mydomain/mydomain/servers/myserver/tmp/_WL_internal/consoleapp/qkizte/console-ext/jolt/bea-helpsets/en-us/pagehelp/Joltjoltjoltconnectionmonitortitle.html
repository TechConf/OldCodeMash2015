You can use this page to view runtime statistics for the connection in this connection pool.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.

Name Description
Address

The connection Address.

MBean Attribute:
JoltConnectionRuntimeMBean.Address

Last Access Time

The Last Access Date and Time.

MBean Attribute:
JoltConnectionRuntimeMBean.LastAccessTime

Alive

The Connection Alive indicator.

MBean Attribute:
JoltConnectionRuntimeMBean.Alive

Requests

The Request Count.

MBean Attribute:
JoltConnectionRuntimeMBean.RequestCount

Pending Requests

The Pending Request Count.

MBean Attribute:
JoltConnectionRuntimeMBean.PendingRequestCount

Errors

The Request Error Count.

MBean Attribute:
JoltConnectionRuntimeMBean.ErrorCount

In Transaction

The Connection in Transaction indicator.

MBean Attribute:
JoltConnectionRuntimeMBean.InTransaction
