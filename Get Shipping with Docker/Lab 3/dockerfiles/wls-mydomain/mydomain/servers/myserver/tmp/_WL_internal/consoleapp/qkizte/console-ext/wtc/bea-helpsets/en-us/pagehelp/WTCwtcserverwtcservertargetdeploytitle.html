WTC Server: Targets

Configuration Options     Related Tasks     Related Topics

Use this page to select the server instance on which you would like to deploy this WTC server. You can reconfigure deployment targets later if you wish.

Targeting deploys the connectivity information contained in a WTC server configuration to a selected server instance. Although you may have many WTC server configurations in the current WebLogic Server domain, only one WTC server configuration can be targeted to any given server instance.

Deciding when to target a WTC server is very important. Once a WTC server is targeted, the configuration deployed to the selected server instance is static. Although you can continue to make changes in your WTC server configuration, any changes made after the configuration is deployed will not be reflected in the selected server instance. You will need to remove the service from the server instance and target the service to the server instance for these changes to take effect.

Configuration Options

Name Description
Target

The list of all WebLogic Server instances in the current domain that are candidates for hosting this WTC server.

MBean Attribute:
WTCServerMBean.Targets
