Coherence Cluster: Configuration: Address Providers

Column Display     Related Tasks     Related Topics

This page displays all the address providers for a Coherence cluster.

Address providers allow Proxy service, remote cache, and remote invocation services to reference configured TCP socket address combinations by name instead of explicitly embedding an addresses in a Cache Configuration file.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.
