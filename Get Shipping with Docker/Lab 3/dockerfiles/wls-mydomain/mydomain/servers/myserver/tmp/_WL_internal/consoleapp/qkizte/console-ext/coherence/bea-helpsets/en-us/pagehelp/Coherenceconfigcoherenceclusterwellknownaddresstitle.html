Coherence Cluster: Configuration: Well Known Address

Configuration Options     Related Tasks     Related Topics

Use this page to configure a well known address for this Coherence cluster. Other members can use this address to enroll in the cluster.

Configuration Options

Name Description
Name

A unique name for this Coherence cluster address.
Listen Address

The IP address.

MBean Attribute:
CoherenceClusterWellKnownAddressBean.ListenAddress

Changes take effect after you redeploy the module or restart the server.

Listen Port

The listen port.

MBean Attribute:
CoherenceClusterWellKnownAddressBean.ListenPort

Minimum value: 1

Maximum value: 65535

Changes take effect after you redeploy the module or restart the server.
