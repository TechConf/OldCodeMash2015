WTC Server: Configuration: Queuing Bridge

Related Tasks     Related Topics

A Tuxedo queuing bridge provides a bidirectional JMS interface that allows WTC applications to communicate with Tuxedo application environments.

This page summarizes the Tuxedo queuing bridge that has been configured in the current WTC server. You can have only one Tuxedo queuing bridge configured. You must delete an existing Tuxedo queuing bridge before attempting to create a new Tuxedo queuing bridge.
