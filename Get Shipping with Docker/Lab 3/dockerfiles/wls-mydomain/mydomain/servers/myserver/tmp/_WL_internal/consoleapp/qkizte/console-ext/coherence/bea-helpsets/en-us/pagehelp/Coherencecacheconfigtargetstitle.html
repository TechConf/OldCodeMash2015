Cache Configuration: Targets

Configuration Options     Related Tasks     Related Topics

Use this page to select the servers or clusters on which you want to deploy this cache configuration.

Configuration Options

Name Description
Independent Servers

Lists the servers in your configuration on which you can deploy the cache configuration. Select servers and click Save.
Clusters

Lists clusters and comprising servers on which you can deploy the cache configuration. Select the cluster on which you want to deploy the cache configuration and click Save. In almost all cases, you should deploy the cache configuration to an entire cluster, not individual servers in a cluster.
