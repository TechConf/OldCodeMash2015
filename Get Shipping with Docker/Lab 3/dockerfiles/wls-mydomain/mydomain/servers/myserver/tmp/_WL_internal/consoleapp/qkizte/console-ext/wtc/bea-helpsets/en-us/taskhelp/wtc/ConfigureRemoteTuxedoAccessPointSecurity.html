Configure security for remote access points


To define the security configuration of a remote Tuxedo access point that will be used with this WTC server:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to configure security for remote access points.
  4. Select Configuration > Remote APs.
  5. On the WTC Remote Access Points page, select the name of the remote access point you want to configure.
  6. Select Security.
  7. Modify the security attributes for your environment.

    See Configuration Options.

  8. Click Save.
  9. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
