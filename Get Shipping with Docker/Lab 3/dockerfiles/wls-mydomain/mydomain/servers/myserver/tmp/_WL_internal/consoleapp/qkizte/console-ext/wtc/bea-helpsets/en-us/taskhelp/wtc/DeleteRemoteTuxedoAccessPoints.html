Delete remote access points


Remote Tuxedo access points provide configuration information to connect a WTC server to available remote Tuxedo domains. To delete a remote Tuxedo access point:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to delete remote access points.
  4. Select Configuration > Remote APs.
  5. On the WTC Remote Access Points page, select the checkbox next to the remote access point you want to delete.
  6. Click Delete.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
