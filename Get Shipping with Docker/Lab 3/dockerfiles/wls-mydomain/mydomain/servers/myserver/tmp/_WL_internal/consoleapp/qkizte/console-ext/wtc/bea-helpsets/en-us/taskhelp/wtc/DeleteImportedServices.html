Delete imported services


Imported services provide information on how to access services that are available on remote Tuxedo domains. To delete an imported service:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to delete imported services.
  4. Select Configuration > Imported.
  5. On the WTC Imported Services page, select the checkbox next to the imported service you want to delete.
  6. Click Delete.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
