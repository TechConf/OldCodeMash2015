Resume WTC services


The WebLogic Tuxedo Connector (WTC) provides interoperability between WebLogic Server applications and Tuxedo services. It allows WebLogic Tuxedo Connector clients to invoke Tuxedo services and Tuxedo clients to invoke WebLogic Server applications, such as EJBs and servlets.

To resume a WTC service that has been suspended:

  1. In the Administration Console, expand Interoperability and select WTC Servers.
  2. On the WTC Servers page, select the name of the WTC server you on which you want to resume WTC services.
  3. Select Control > Services.
  4. On the Services page, select the checkbox next to a service with a status of Suspended.
  5. Click Resume to make the advertised service available to WTC servers. The status of the service should change to Available.
