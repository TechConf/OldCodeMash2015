Queuing Bridge: Factories

Configuration Options     Related Tasks     Related Topics

Use this page to define the configuration of connection factories for use with this Tuxedo queuing bridge.

Configuration Options

Name Description
JNDI Factory

The name of the JNDI lookup factory.

Example: weblogic.jndi.WLInitialContextFactory

MBean Attribute:
WTCtBridgeGlobalMBean.JndiFactory

JMS Factory

The name of the JMS connection factory.

Example: weblogic.jms.ConnectionFactory

MBean Attribute:
WTCtBridgeGlobalMBean.JmsFactory

Tuxedo Factory

The name of the Tuxedo connection factory.

Example: tuxedo.services.TuxedoConnection

MBean Attribute:
WTCtBridgeGlobalMBean.TuxFactory
