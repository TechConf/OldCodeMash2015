Configure connections for queuing bridges


To define the connection configuration for this Tuxedo queuing bridge:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the Administration Console, expand Interoperability and select WTC Servers.
  3. On the WTC Servers page, select the name of the WTC server on which you want to configure connections for queuing bridges.
  4. Select Configuration > Queuing Bridge.
  5. On the WTC Tuxedo Queuing Bridges page, select the name of the queuing bridge you want to configure.
  6. Select Connections.
  7. Modify the connection attributes for your environment.

    See Configuration Options.

  8. Click Save.
  9. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
