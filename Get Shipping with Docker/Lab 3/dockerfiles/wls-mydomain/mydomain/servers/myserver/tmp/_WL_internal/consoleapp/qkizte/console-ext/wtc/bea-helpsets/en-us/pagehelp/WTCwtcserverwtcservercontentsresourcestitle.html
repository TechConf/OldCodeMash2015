WTC Server: Configuration: Resources

Related Tasks     Related Topics

Resources specify field table classes, reference view buffer structures, and provide application passwords for domains.

This page summarizes the resource that has been configured in the current WTC server. You can have only one resource configured. You must delete an existing resource before attempting to create a new resource.
