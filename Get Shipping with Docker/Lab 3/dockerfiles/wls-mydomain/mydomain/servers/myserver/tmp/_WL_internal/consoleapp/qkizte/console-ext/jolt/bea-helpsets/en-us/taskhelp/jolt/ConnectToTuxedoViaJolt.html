Connect to Tuxedo with Jolt

Before you begin

  • Configure Oracle Tuxedo to use Jolt.
  • Configure applications or servlets to use Jolt APIs.

Jolt is a Java-based client API that manages requests from servlets and applications on WebLogic Server to Oracle Tuxedo services.

To enable servlets and applications to connect to Oracle Tuxedo using Jolt:

  1. Configure Jolt startup and shutdown classes
  2. Create Jolt connection pools

    Oracle recommends that you create one Jolt connection pool for each application running on WebLogic Server.

  3. Target Jolt connection pools
