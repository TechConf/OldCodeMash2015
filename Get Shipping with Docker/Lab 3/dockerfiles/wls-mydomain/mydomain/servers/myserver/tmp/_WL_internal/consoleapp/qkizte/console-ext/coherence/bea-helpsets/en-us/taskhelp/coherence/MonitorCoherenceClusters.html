Monitor Coherence clusters


To monitor run-time information for a Coherence cluster associated with a particular application or module:

  1. In the left pane of the Console, select Deployments.

    The Deployments table lists all the deployments currently installed on WebLogic Server. The Type column specifies whether a deployment is an enterprise application, a Web application, or an EJB module.
  2. In the Deployments table, select the name of the enterprise application for which you want to monitor Coherence clusters.
  3. Select Monitoring > Coherence.

    The Coherence Clusters table displays run-time information for Coherence clusters associated with the application or module. For more information, see:

    • Deployments: Applications: Monitor: Coherence
    • Deployments: EJB: Monitor: Coherence
    • Deployments: Web Applications: Monitor: Coherence
