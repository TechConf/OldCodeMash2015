Configure Coherence clusters


To configure existing Coherence clusters:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane of the Console, expand Environment and select Coherence Clusters.

    The Summary of Coherence Clusters page displays the Coherence cluster configurations that have been created in this domain.
  3. Select the name of the cluster you want to configure.
  4. On the Configuration: General page, adjust the cluster properties values as needed.

    For more information about these fields, refer to Coherence Cluster: Configuration: General.

  5. When you are finished making changes, click Save.
  6. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
